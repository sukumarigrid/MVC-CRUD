# MVC CRUD Student Manager

This is a simple console-based MVC CRUD project for assignment use.

## Structure

- `model` - `Student`
- `repository` - in-memory storage and lookup
- `service` - business rules and validation
- `view` - console prompts and output
- `controller` - menu flow and action handling

## Run

```bash
mvn test
mvn exec:java
```

## Menu

1. Create student
2. View all students
3. View student by id
4. Update student
5. Delete student
6. Exit
